# Changelog

## 0.1.1 - 2026-03-01

- Extension defaults now use the production backend URL (`https://chromefactcheck-api.azurewebsites.net`) for first-time installs.
- Trial token defaults are aligned to `20000` in backend configuration and options.
- Trial-expired (`402`) backend response text is localized for `en-US`, `sv-SE`, `de-DE`, `fr-FR`, and `es-ES` with English fallback.
